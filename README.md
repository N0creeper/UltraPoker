## Installation
Ce fichier explique comment installer et exécuter UltraPoker, ainsi que les contraintes techniques nécessaires.

### Version de Python requise
UltraPoker nécessite **Python 3.10 - 3.13**.

#### Bibliothèques
Nécessite :
- `pygame>=2.5.0`

Pour installer :
```bash
pip install pygame
```

### Lancer UltraPoker
Ouvrir le fichier `Sources/main.py`.

Ou dans le terminal de commande :
- Aller dans le dossier du projet :
  ```bash
  cd /path/to/UltraPoker
  ```
- Puis lancer :
  ```bash
  python Sources/main.py
  ```

# Informations
Projet réalisé pour les **Trophées NSI 2026**.

## Objectif du projet
- Créer un jeu de poker accessible et agréable à jouer.
- Développer une IA plus réaliste et moins prévisible.
- Concevoir un moteur de jeu complet et fiable.
- Proposer une interface graphique personnalisée.

## Fonctionnalités principales
- Moteur de jeu complet (distribution, mises, tours, showdown).
- Algorithme de détection des mains (paire, couleur, full, etc.).
- IA basée sur des ranges et des décisions stratégiques.
- Interface graphique réalisée sur mesure.
- Tests manuels et retours de testeurs externes.
